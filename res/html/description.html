About HAPIEST


HAPIEST is a Graphical User Interface (GUI) made to allow access to spectroscopic data stored in the HITRAN database without having knowledge of python programming.


HAPIEST was developed in a software engineering course at the State University of New York at Oswego by students Benji Caro, Joshua Karns, Dominik Lohmann, Wyatt Matt, Ethan Messer, and Michael Sova in conjunction with Dr. Iouli Gordon and Dr. Roman Kochanov of Harvard-Smithsonian Center for Astrophysics, and with advisement from SUNY Oswego professors Bastian Tenbergen and Shashi Kanbur.


HAPIEST takes advantage of the HITRAN Application Programming Interface Python library (HAPI):


R.V. Kochanov, I.E. Gordon, L.S. Rothman, P. Wcislo, C. Hill, J.S. Wilzewski, HITRAN Application Programming Interface (HAPI): A comprehensive approach to working with spectroscopic data, J. Quant. Spectrosc. Radiat. Transfer 177, 15-30 (2016) [DOI:10.1016/j.jqsrt.2016.03.005].

Features


Fetch Tab

The fetch tab is where you can download data from HITRAN. Any fetched data can subsequently be used to make graphs, or it can be modified.

Molecule Tab

The molecule tab displays information about the molecules HAPIEST can work with.

Edit Tab

The edit tab allows you to view and modify tables like a spreadsheet.

Graphing Tab

The graphing tab is where you can turn the data downloaded from HITRAN into graphs, including absorption coefficient, transmittance spectrum, radiance spectrum, and absorption spectrum graphs.

Graph Display Window

The graph display window is where you can view and compare graphs. Multiple graphs (of the same type) can be sent to the same window. The coordinates from the generated graphs can be downloaded as a CSV file, plain text file, or JSON file.
